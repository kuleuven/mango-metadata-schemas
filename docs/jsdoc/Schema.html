Name Type Description

container String ID of the DOM element inside which this schema version is displayed.
urls UrlsList Collection of URLs and other information obtained from the server side.
parent String If this schema was born as clone of another schema, the name and version of the parent schema.
origin Object If this is a draft initiated as clone of a published schema, information on the fields that are cloned with it.

form SchemaDraftForm Form that gets submitted when the schema version is saved or published.
