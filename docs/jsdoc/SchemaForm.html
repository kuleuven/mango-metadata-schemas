Class: SchemaForm

SchemaForm(json, container_id, prefix)

Class for a published version of a schema to be used when applying metadata.

Constructor

new SchemaForm(json, container_id, prefix)

Create a form to apply metadata from a schema.
Parameters:
Name Type Description
json SchemaInfo Contents of the JSON file on which the schema is stored.
container_id String ID of the DIV to which the form will be attached.
prefix String Prefix of the metadata attribute names, e.g. `msg.book`.
Properties:
Name Type Description

realm String Name of the realm to which the schema belongs.

field_ids Array.<String> Ordered list of IDs of the fields.
