Class: NavBar

NavBar(id, extra_classes)

Class to create a navigation bar with pills or tabs. This is used to display the different versions of a schema, with tabs that have badges, and to switch between views and editors within a version of a schema.

Constructor

new NavBar(id, extra_classes)

Initialize a navigation bar to host versions of a schema or view/editors of a version.
Parameters:
Name Type Description
id String ID of the NavBar instance, used for the IDs of the navigation list and contents.
extra_classes Array.<String> Names of classes to be added to the list of navigation bar, e.g. 'justify-content-end' and 'nav-pills'.
Properties:
Name Type Description
nav_bar HTMLUListElement A list of tabs for navigation.
tab_content HTMLDivElement The content of the tabs
id String ID of the instance, used for the IDs of its DOM components.
