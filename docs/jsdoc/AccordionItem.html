Class: AccordionItem

AccordionItem(id, header_title, accordion, is_new)

Class to create a standard BS5 Accordion Item, in which the information of a given schema and all its versions will be displayed.

Constructor

new AccordionItem(id, header_title, accordion, is_new)

Initialize a new accordion item to host a schema and all its versions.
Parameters:
Name Type Default Description
id String ID of the accordion item.
header_title String Title of the accordion item, i.e. the name of the schema it shows.
accordion String ID of the accordion itself.
is_new Boolean false Whether the item corresponds to the "new schema" editor.
Properties:
Name Type Description

div HTMLDivElement The accordion item itself.
