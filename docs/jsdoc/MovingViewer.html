Class: MovingViewer

MovingViewer(form, schema)

Class representing box to view and edit an input field when editing a schema.

Constructor

new MovingViewer(form, schema)

Create a box to show and allow editing / placement / removal / duplication of a field.
Parameters:
Name Type Description
form InputField Field to be edited.
schema ComplexField Schema or mini-schema of a composite field to which the field belongs.
Properties:
Name Type Description
title String Title of the box (title of the field, with asterisk if required).
repeatable Boolean Whether the field is repeatable.
div HTMLElement Box containing the title, example and buttons.
body HTMLElement Example input field, showing what the field looks like.
parent_modal bootstrap.Modal If the field is inside a composite field, the composite field's editor modal.
rem HTMLButtonElement Button used to remove the field.
copy HTMLButtonElement Button to duplicate the field.
edit HTMLButtonElement Button to edit the field.
schema ComplexField The schema (or mini-schema of composite field) to which the field belongs.
below HTMLButtonElement Button under the field used to add a new field under it. It moves along with the field.

duplicate(form)

Create a duplicate of a field with empty ID. Until the duplicate has a new ID, the (mini-)schema cannot be saved or published. The duplicate itself cannot be copied (the button is disabled) and the "edit" button is highlighted.
