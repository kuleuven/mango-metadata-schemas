(static) ask_confirmation(body, action, dismiss)

Fill in the existing confirmation modal and show it to obtain a simple yes/no answer to a confirmation question (e.g. Are you sure you want to delete this field?). This does not apply to modals created with the Modal class, but refers to a modal.
Parameters:
Name Type Description
body String Descriptive text to append to the modal (what are the consequences of accepting this?).
action function What to do after a positive answer.
dismiss function What to do after dismissal of the modal (if anything).

(static) submit_confirmation(body, url, form_data, extra_Action)

Fill in the existing confirmation modal and its form and show it to obtain a simple yes/no answer to a confirmation question (e.g. Are you sure you want to discard this draft?). This does not apply to modals created with the Modal class, but refers to a modal. The confirmation button becomes now a submission button for the form contained in the modal.
Parameters:
Name Type Description
body String Descriptive text to append to the modal (what are the consequences of accepting this?).
url String URL to post the contents of the hidden form to.
form_data Object.<String, String> Names and values of the hidden fields to add to the form.
extra_Action function Something extra to do on submission, if relevant.
