Class: ComplexField

ComplexField(name, data_status)

Master class to represent schemas and mini-schemas. Only the child classes are actually instantiated.

Constructor

new ComplexField(name, data_status)

Initialize a Schema, mini-schema for a composite field or dummy mini-schema for illustrating a composite field.
Parameters:
Name Type Default Description
name String Initial name of the schema, for IDs of the DOM elements.
data_status String draft Derived status used in IDs of DOM elements.
Properties:
Name Type Description
modal_id String ID of the modal through which a new input field can be chosen.
initial_name String Placeholder name for DOM element IDs.
title String User-facing label of the schema (or composite field).
data_status String Derived status used for IDs of DOM elements, e.g. 'new', 'copy', 'draft'...
initials Object.<String, InputField> Collection of empty fields to start creating.
Properties
Name Type Description
typed TypedInput Initial simple field.
select SelectInput Initial single-value multiple-choice field.
checkbox CheckboxInput Initial multiple-value multiple-choice field.
object ObjectInput Initial composite field.
field_ids Array.<String> Ordered names of the fields.
fields Object.<String, InputField> Collection of fields that belong to the schema.
properties Object.<String, FieldInfo> Object-version of the information of the fields, to store in a JSON.
new_field_idx Number Index of the following field that could be added.
